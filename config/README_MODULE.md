# Config Directory

Static configuration files for the QuantLab Intelligence Platform.

## Files
| File | Purpose |
|------|---------|
| `sector_context.json` | Sector → signals mapping for focus list |
| `institutional_calendar.json` | FOMC, OPEX, WASDE, etc. |
| `shareholder_letters.json` | CEO letter calendar |
| `ag_basket.json` | Upstream/mid/downstream ag tickers |
| `pytrends_queries.json` | AI cascade search terms (15 layers) |
